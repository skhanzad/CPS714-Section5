# CPS714-Section5
LibraLite includes the following key features:
Catalog Page: Displays the library’s items in a clean, organized layout.
Search Engine: Search by title, author, or genre with fast and accurate results.
Filters: Narrow results by availability, format, or genre.
Item Tags: Quickly identify categories such as “Classic Literature” or “English.”
Item Details Page: View full information including description, location, and status.

To access the catalog:
Ensure Node.js is installed.
Navigate to the project directory that contains package.json.
Run:
npm install
npm run dev
Open your browser and go to:
 http://localhost:5173/
The catalog will load automatically.
